About VisuLeMo :

A person(user) can have interests from the various areas such as social, education, entertainment etc.. From the such details of a person, we can create person(user)'s model called a learner model.

There are already developed frameworks(learner models) such as

  • PALM : which gives academic interests of the user
  • webtrace : which gives social interests of the user
  • and other frameworks like these can be developed and added.

So, the framework Visualizations of Learner Models (VisuLeMo) will fetch user's interests from these different frameworks and visualize them. The aim is to see whether these visualizations help user to support awareness and self-reflection.

VisuLeMo has been developed at the i9 Learning Technologies Research Group of RWTH Aachen University.
